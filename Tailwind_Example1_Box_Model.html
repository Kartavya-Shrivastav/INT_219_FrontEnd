Tailwind CSS Box Model Example

Box Model

This box demostrates the properties of the box model using Tailwind CSS:

Padding

This box has p-4, which means padding is 1 rem (16px).

Border

This box has border-4, a dashed border with border-blue-300.

Margin

This box is separated from other using mb-4 (margin-bottom).

Width and Height

This box has w-64 (16 rem) and h-32 (8 rem).
